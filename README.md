# Vibe-Checking-First-JS-App-
A lightweight web application that analyzes user input to determine the "vibe" using JavaScript logic.

How to Run Locally To get this project up and running on your machine, follow these steps:

1-Clone the repository or download the source code.

2-Navigate to the project folder.

3-Open index.html directly in your web browser (Chrome, Firefox, or Safari).

Alternative: If you are using VS Code, right-click index.html and select "Open with Live Server" for a real-time preview.
